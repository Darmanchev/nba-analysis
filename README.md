# Анализ данных NBA с помощью машинного обучения

Этот проект представляет собой анализ данных игроков Национальной Баскетбольной Ассоциации (NBA). Главная цель проекта — показать, как можно использовать историческую статистику и алгоритмы машинного обучения для предсказания роли и будущего баскетболистов.

Проект написан так, чтобы быть понятным даже без глубоких знаний в программировании и сложной математике.

## Что внутри?

Проект разделен на две основные части (два разных исследования):

### 1. Угадываем позицию игрока на поле
**Файлы:** `main.ipynb` и `NBA_Random_Forest_Notebook.ipynb`

Здесь мы учим программу определять игровую позицию баскетболиста (например, разыгрывающий, центровой, форвард) исключительно по его физическим данным (рост, вес) и статистике в матчах (очки, подборы, передачи).
- Мы используем метод под названием "Случайный лес" (Random Forest). Если говорить просто, компьютер задает много-много вопросов ("рост больше 2 метров?", "подборов больше 5?") и на основе ответов делает вывод.
- Программа анализирует данные и сама находит закономерности — например, что высокие и тяжелые игроки, делающие много подборов, чаще всего оказываются "центровыми".

### 2. Предсказываем будущее новичков
**Файл:** `rookie_prediction.ipynb`

Можем ли мы понять, станет ли молодой игрок звездой, глядя только на его первые два сезона в лиге? 
В этой части мы решаем именно эту задачу:
- Берем статистику игроков только за их первый и второй год в NBA.
- Пытаемся предсказать, к какой из трех групп они будут относиться в будущем:
  - **Вылетят из лиги** (не дойдут даже до 3-го сезона).
  - **Станут ролевыми игроками** (будут играть, но набирать в среднем меньше 12 очков за игру).
  - **Станут звездами или игроками основы** (будут стабильно набирать больше 12 очков за игру).
- Проверка делается очень честно: алгоритм никак не "подглядывает" в будущее игроков, когда делает свои предсказания.

## Откуда данные?
Данные берутся из открытых источников на платформе Kaggle. Они включают:
- Данные о физических характеристиках игроков.
- Историческую статистику по всем сезонам (очки, подборы, передачи, блоки и т.д.).

Файлы с данными должны лежать в папке `data/`.

## Как посмотреть и запустить?
Все исследования оформлены в виде "интерактивных тетрадей" (Jupyter Notebook) — это формат, где код перемешан с текстом, объяснениями и красивыми графиками.

1. Убедитесь, что у вас установлен Python и Jupyter (если вы запускаете код на своем компьютере).
2. В папке `data/` лежат файлы с данными. (В самих файлах проекта также есть скрипт для их автоматического скачивания из интернета).
3. Откройте любой из файлов с расширением `.ipynb` и запускайте ячейки по очереди. 

Внутри файлов много наглядных графиков и таблиц, которые показывают, как именно алгоритм принимает решения и насколько часто он оказывается прав.
